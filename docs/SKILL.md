# Habilidades do Codex Assistant

## Capacidades Atuais
- **Análise de Código**: Capacidade de entender arquivos Python e sugerir refatorações.
- **Explicação de Arquitetura**: Resumir e explicar o fluxo de dados baseado no `ARCHITECTURE.md`.
- **Geração de Boilerplate**: Criar novos módulos seguindo os padrões de `RULES.md`.

## Em Desenvolvimento
- **Multi-linguagem**: Suporte além do Python (JS/TS, Go).
- **RAG (Retrieval Augmented Generation)**: Para indexação de grandes bases de conhecimento.
- **Git Integration**: Capacidade de sugerir commits e ler o histórico do Git.
